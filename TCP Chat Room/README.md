A simple **TCP-based chat room** built in Python using sockets and threading.  
This project allows multiple clients to connect to a server and chat in real time.

---

## Features

- Multiple clients can join the chat room simultaneously
- Each client has a nickname
- Messages are broadcasted to all connected clients
- Clients are notified when someone joins or leaves
- Built with Python's `socket` and `threading` modules

---

## How to Run

### 1. Start the Server
```bash
python server.py
```
### 1. Start the Client
```bash
python client.py
```
